bronto arduino circuit

a brontosaurus shaped lighting display using an Arduino Nano V3, six 74HC595 serial-in-parallel-out shift registers driving 48 LEDs!

the source code that's uploaded to the arduino is written entirely in c without the arduino framework - it interacts directly with the registers on the ATmega328P microcontroller
