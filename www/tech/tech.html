supporting technology - jdeko.me

/-|-\

docker

each software element that supports this website has been containerized with docker. a single docker compose file orchestrates the startup and teardown of every container, network, and volume

cloudflare tunnel

a cloudflare tunnel is used to securely route https traffic through my local server. a cloudflared server is running in a container to route the https requests locally

nginx web server

an nginx web server serves static files and reverse-proxies requests for /bball to the jdeko.me/bball api

postgres database

a postgres database serves nba/wnba stats to the jdeko.me/bball api - the stats are extracted from the official nba api via a command-line-interface tool i wrote in go

javascript/typescript

javascript transpiled from typescript powers the frontend of jdeko.me/bball. all interactions with any of the buttons/selectors/text-boxes on this page are handled by javascript event handlers that call endpoints from my API

html/css

all visual elements of this website are written in pure html & css. i've always preferred clean and to-the-point websites, and i miss the more simplistic look of the internet of my youth
